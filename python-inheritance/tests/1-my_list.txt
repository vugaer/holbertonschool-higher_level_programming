The MyList class
=================

Importing the class:
    >>> MyList = __import__('1-my_list').MyList

Test: Basic functionality with positive integers
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Test: Empty list
    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []

Test: Single element
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.print_sorted()
    [1]

Test: Already sorted list
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.print_sorted()
    [1, 2, 3]

Test: Reverse sorted list
    >>> my_list = MyList()
    >>> my_list.append(5)
    >>> my_list.append(4)
    >>> my_list.append(3)
    >>> my_list.append(2)
    >>> my_list.append(1)
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Test: List with negative numbers
    >>> my_list = MyList()
    >>> my_list.append(-5)
    >>> my_list.append(1)
    >>> my_list.append(-3)
    >>> my_list.append(10)
    >>> my_list.print_sorted()
    [-5, -3, 1, 10]

Test: List with duplicates
    >>> my_list = MyList()
    >>> my_list.append(3)
    >>> my_list.append(1)
    >>> my_list.append(3)
    >>> my_list.append(2)
    >>> my_list.print_sorted()
    [1, 2, 3, 3]
